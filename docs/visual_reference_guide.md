## Print Layout

```
When printing, the report auto-adjusts:

✓ Cards: 2 columns on print
✓ Charts: Full page width
✓ Tables: Compact font, all columns visible
✓ Colors: Print-friendly (darker shades)
✓ Page breaks: Between major sections
✓ Header/Footer: Company name, date, page numbers
```
